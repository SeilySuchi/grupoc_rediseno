== Primary Menu ==

- "Primary Menu" will work best for six to eight menu items at top level.
- "Primary Menu" supports Five level nested menu depth including top level.

== Help ==

See the "Help" tab in the "Prana Options" admin page for license, changelog and other informations.

== Installation ==

Installing DesignOrbital Theme From WordPress Admin Panel

1. Go to `Appearance > Themes`
2. Click `Add New`
3. Click `Upload`
4. Choose File (Downloaded ZIP File) > Install Now
5. Click `Activate`

Theme Options

1. See `Appearance -> Bandana Options` for Theme Options
